=============
Raman Tablets
=============

Overview
########
Active substance in pharmaceutical tablets.

Name
####
Raman Tablets

Id
##
`raman_tablets`

Description
###########
.. note::
    - NIR FT Raman; 3401 variables; 3600 200 cm 1
    - Relative active substance content (% w/w)
    - Tablet type (A, B, C, D)

:No. of samples:
    120
:No. of features:
    3401 attributes, 1 attributes (`Type`) associated with classes, 1 property (`% w/w`)
