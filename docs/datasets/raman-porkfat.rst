=============
Raman Porkfat
=============

Overview
########
Raman spectral data set of pork fat samples.

Name
####
Raman Pork Fat

Id
##
`raman_porkfat`

Description
###########
This data set contains Raman information of pork fat samples.

:No. of samples:
    105
:No. of features:
    5667 attributes, 3 attributes associated with classes, 19 properties
